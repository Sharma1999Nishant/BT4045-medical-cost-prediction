Open this Projectfolder

Step-1) Install - python and all its library 

Step-2) Once you're installed python , restart your device.

Step-3) Open this project folder and run command prompt .

Step-4) Type python v for checking the version of python .

Step-5) After checking the version type " app.py " , then command prompt give you a local address like this - "http://127.0.0.1:5000/".

Step-6) Open any installed browser and type that local address and enter that.

Step-7) The browser will open a window and form and then fill form and give the details required and press "Predict probability".

Step-8) Your output will be display on screen.
